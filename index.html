Racing & sports · syndicate side · model → price → size

An edge is a disagreement with the market, priced.

A bookmaker sells you a margin. A syndicate does the opposite: it builds an independent probability, finds where the market is wrong, and stakes the gap. This terminal runs that whole pipeline live — strip the vig, rate the field, price the exotics, size with Kelly, and account for the fact that in a pool your own money moves the odds. Edit the race below; every number recomputes.

Strip the margin

02

Your opinion

Ratings become probabilities through a tempered softmax (multinomial-logit / Luce choice): p_i = e^(β·r_i) / Σ e^(β·r_j). β is conviction — flat field at 0, all-in on the top rating as β grows. Then shrink toward the crowd, because a lone opinion is noisy.

At 0% shrink you bet pure model. Pushing right blends your probabilities geometrically toward the devigged market — the same move a desk makes before firing, trading edge for robustness.

03

Price the exotics

Win probabilities imply the whole finishing order via Harville / Plackett-Luce: draw each place proportional to remaining strength. Harville over-rates the favourite's place chance, so a discount exponent λ<1 flattens the back end toward Henery/Stern.

05

The pool fights back

Against a tote, the dividend is set by where the money lands — D(s) = (1−t)(T+s)/(W+s) — so your own stake dilutes your own price. There is a finite optimal bet where the marginal edge hits zero. This is the game-theoretic core of pool betting, and why naive fixed-odds Kelly over-bets.

What's under the hood

Every number on this page is computed live in the browser from the same equations as the tested Python core that ships with the repo (core/ + tests/, 25 property tests). No data is sent anywhere; the terminal runs offline. Built as a study of the mathematics a proprietary syndicate uses to find and size an edge.
